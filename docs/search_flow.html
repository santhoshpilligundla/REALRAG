🔍 What Happens When You Search

End-to-end journey of a question in RealRAG

⚡ 4-Tier Auto-Escalation

Tier 1 — Fast
Instant retrieval + LLM answer
Tier 2 — Deep
More chunks, wider search
Tier 3 — Agent
Multi-step tool-use loop
Tier 4 — Live
Headless Claude agents on source
💬
User Types a Question
User types a question in the chat box and hits Enter. The question is saved to the session for conversation context.
frontend/streamlit_app.py lib/sessions_repo.py
1
🔑
Cache Check
Before any LLM call, the exact question is checked in the query cache. If a previous identical question was answered, return it instantly (0 API cost).
lib/query_cache.py
~0ms cache hit
2
🧠
Embed the Question
The question is sent to OpenAI's text-embedding-3-large model to create a 3072-dimension vector representing its semantic meaning.
lib/embedder.py
3
🔎
5-Method Parallel Retrieval
Five search methods run simultaneously:

① Vector — FAISS semantic similarity
② Keyword — Postgres full-text search
③ Symbol — Exact name match in DB
④ KG — Knowledge graph SQL triples
⑤ Doc-vec — Generated doc embeddings
lib/retrieval.py lib/faiss_store.py lib/kg.py
4
🏆
Re-rank & Select Top Hits
Top 50 hits from all methods are deduplicated and re-ranked by score. Top 5 are selected and formatted into a context window for the LLM.
lib/retrieval.py
5
🤖
LLM Generates Answer
Claude (Haiku by default) is given the context + question with a strict system prompt: answer in plain business English, no code references, no class names, cite only from the provided context.
lib/chat.py lib/llm.py
6
✅
Faithfulness Check
The answer is automatically validated: Does it look uncertain? Does it mention code artifacts (hallucination)? Do the named subjects appear in retrieved context? If weak → escalate to the next tier.
lib/chat.py
Escalates to Tier 2 if weak
7
📤
Answer Returned to User
The final answer is streamed to the UI as plain business English. Citations (file, entity, line numbers) are shown in a collapsible "Sources" section. Answer is saved to session + cache.
lib/sessions_repo.py lib/query_cache.py
8
